INSTALL
-------

1. Copy the urlclass.module to the Drupal modules/ directory.

2. Go to administer/modules, and enable the urlclass.module

3. Go to admin/filters and enable the filter in the desired input formats.

4. Optionally rearrange the filters on the input format's 'rearrange' tab to resolve conflicts with other filters.
